Tabs & Tab Size

Your text editor should insert spaces for all languages. No tabs allowed here! 4 spaces per tab for all languages.

"tab_size": 4,
"translate_tabs_to_spaces": true,

SublimeLinter

PEP8 - Style Guide for Python Code

http://legacy.python.org/dev/peps/pep-0008/

  • the yellow rectangles on your source are flags that you're doing something wrong
  • ignore the visual rules
"flake8": {
    ...
    "ignore": "E501, E126, E127, E128",
    ...
},

@ JS: always use strict, should pass jshint (SublimeLinter)

"use strict";

some additional notes regarding our coding style/convention

  • Variables and functions should have logical names.
  • Avoid one letter names where possible (exception: counters or iterators).
  • Never use hard coded (magic) numbers or strings.
  • Make them module-level constants or even better make them settings variables in Ferris.

Naming convention

module_name
package_name
ClassName
method_name
ExceptionName
function_name
GLOBAL_CONSTANT_NAME
global_var_name
instance_var_name
function_parameter_name
local_var_name

Braces should be on the same line as control statements for languages that use braces

if (number < 8) {
